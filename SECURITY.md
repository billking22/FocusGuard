# 安全政策

## 支持的版本

目前，我们仅支持最新版本的 FocusGuard。安全更新将仅适用于最新版本。

## 报告漏洞

如果您发现潜在的安全漏洞，请**不要**公开提交 Issue。

**正确的报告方式：**

1. 发送邮件至：security@example.com（请替换为实际邮箱）
2. 邮件主题格式：`[SECURITY] 漏洞描述`
3. 在邮件中详细描述：
   - 漏洞类型
   - 影响范围
   - 复现步骤
   - 您的联系方式

**我们承诺：**

- 尽快确认收到（通常在 48 小时内）
- 评估严重程度并确定修复计划
- 修复后发布公告
- 在适当的情况下公开致谢

## 安全最佳实践

### 用户应该做的

1. **保持更新**：始终使用最新版本的 FocusGuard
2. **检查来源**：只从官方渠道下载应用
3. **授予最小权限**：仅授予必要的系统权限（摄像头）
4. **保护 API 密钥**：不要分享您的 AI API 密钥
5. **使用本地模型**：如需最高隐私保护，使用 Ollama 本地模型

### FocusGuard 的安全设计

- **隐私保护**：所有图像分析在内存中进行，永不存储到磁盘
- **数据最小化**：仅保存文本统计信息（状态/置信度/时间戳）
- **本地优先**：支持纯本地 AI 模型（Ollama），无需网络连接
- **系统权限**：仅需摄像头权限，不访问其他敏感数据
- **加密传输**：API 密钥使用安全存储，网络请求使用 HTTPS

## 已知的安全问题

目前没有已知的安全漏洞。

## 隐私声明

FocusGuard 是一个隐私优先的应用：

- **摄像头数据**：仅在应用运行时访问，用于专注状态分析
- **图像处理**：在内存中临时处理，不保存到磁盘
- **统计数据**：仅记录时间戳、状态和置信度（纯文本）
- **AI 服务**：使用 HTTPS 加密传输到第三方 AI 服务（可选）
- **本地模式**：使用 Ollama 可实现完全本地化运行

详细信息请查看 [README.md](README.md) 的隐私说明部分。
